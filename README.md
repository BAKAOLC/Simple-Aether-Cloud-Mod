# Simple Aether Cloud Mod

一个简单的Minecraft模组，添加了多种云方块。原始思路来自The Aether模组，在此基础上添加了自己的特性。

## 功能

- 添加了黄云、蓝云、水平蓝云和红云方块
- 云方块可以像普通方块一样放置和破坏
- 云方块具有特殊的碰撞箱，允许玩家穿过
- 红云会对接触的实体造成伤害

## 合成配方

### 黄云

```
糖 + 黄色染料 = 黄云
```

### 蓝云

```
糖 + 蓝色染料 = 蓝云
```

### 水平蓝云

```
蓝云 = 水平蓝云
水平蓝云 = 蓝云
```

水平蓝云会将被弹射的实体推向方块朝向的方向。可以通过无序合成在蓝云和水平蓝云之间互相转换。

### 红云

```
糖 + 红色染料 = 红云
```

## 特性

- 云方块可以像普通方块一样放置和破坏
- 云方块具有特殊的碰撞箱，允许玩家穿过
- 红云会对接触的实体造成伤害
- 云方块在物品栏中显示为3D模型

## 安装要求

- Minecraft 1.21 ~ 1.21.1
- NeoForge 21.0.167 及以上
